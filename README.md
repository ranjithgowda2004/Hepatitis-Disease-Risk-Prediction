## Hepatitis Risk Prediction 

A small machine learning project that predicts the risk level of Hepatitis patients using basic clinical and laboratory data.
Dataset: UCI Hepatitis Dataset
Models: Logistic Regression, SVM, Random Forest, KNN, AdaBoost, XGBoost
UI: Streamlit web app with simple, user-friendly inputs
Goal: Learning ML + deployment, not medical accuracy
